# Galaxy Toolsmith

Galaxy Toolsmith is a library-first toolkit for building Galaxy tool-wrapper datasets, fine-tuning model variants, and generating Galaxy XML wrappers from command help text and optional source context.

Use the CLI (`gtsm`) for workspace setup, corpus extraction, training, evaluation, export, and local/remote generation workflows.

## Scope

- Build and enrich corpus data from `tools-iuc` and related sources.
- Train and manage model variants across local/open-source-first workflows.
- Generate and validate Galaxy wrappers with optional server/client deployment.
